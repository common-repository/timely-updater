== Installation ==

1. Upload `timely-updater` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu item in the WordPress Dashboard.
3. Once the plugin is activated, you can update your All-in-One Event Calendar plugin by going to:
4. IF All-in-One Event Calendar plugin is enabled, there is a new menu item under Events menu -> Timely Updater
4. ELSE All-in-One Event Calendar plugin is not enabled, there is a new menu item under Settings menu -> Timely Updater
5. Clicking on the menu will check the existing version of the plugin. If it is lower than the latest stable version available 
the updater will attempt to upgrade your All-in-One Event Calendar plugin.
If All-in-One Event Calendar plugin is not found, the plugin will not do anything and you will see a message that 
the plugin hasn't been installed. If All-in-One Event Calendar plugin is at the latest version, the updater will tell you that 
you are running the latest version of the plugin.
